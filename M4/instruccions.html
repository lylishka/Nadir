Instruccions del Joc

Com jugar

1. Tria la teva aventura

Selecciona entre Protocol Epsilon (ciència-ficció) o Curtain Call: Zero (terror surrealista)

2. Escull el teu personatge

Cada personatge té habilitats úniques que afecten les probabilitats d'èxit de les teves accions

3. Llegeix amb atenció

La narrativa et presenta situacions crítiques. Cada paraula compta per entendre el context

4. Pren decisions

Tria entre opcions comunes o opcions exclusives del teu personatge per avançar

Mecàniques del Joc

account_tree

Sistema de decisions ramificades

Cada elecció obre nous camins i tanca altres. La història canvia permanentment segons les teves accions

casino

Sistema de probabilitat

Les accions dins la teva especialitat tenen èxit alt. Les accions fora de la teva especialitat poden fallar o tenir conseqüències greus

groups

Opcions exclusives per personatge

Només el teu personatge pot accedir a certes opcions. L'Enginyer repara, el Metge cura, el Soldat combat

alt_route

Múltiples finals

Protocol Epsilon té 3 finals possibles. Curtain Call: Zero també té 3 finals. El teu destí depèn de les teves decisions

block

Sense punts de guardada

Les decisions són permanents. Un cop acabat el joc, no pots tornar enrere. Cada partida és única i definitiva

Característiques de les Aventures

rocket_launch

Protocol Epsilon

Ciència-ficció. Nau espacial. IA misteriosa. 4 personatges jugables. 3 finals possibles. Confia en la màquina o mor intentant-ho.

theater_comedy

Curtain Call: Zero

Terror surrealista. Teatre infinit. Realitat i ficció es confonen. 3 personatges. 3 finals. Desperta o queda't per sempre.

percent

Probabilitats

Les accions dins la teva especialitat tenen èxit garantit. Fora d'ella, pots fallar i obrir noves rutes narratives.

tips_and_updates

Consells

Llegeix amb atenció. Pensa bé abans de decidir. No hi ha marxa enrere. Cada partida és definitiva i única.
